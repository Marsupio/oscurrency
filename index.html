Welcome to oscurrency.

This project helps small communities match unmet needs with available resources. Installation on heroku is simple:

$ git clone git://github.com/oscurrency/oscurrency.git
$ edit config/config.yml
$ script/heroku_install

If you're a developer interested in contributed to the latest, there is a rails31 branch and more.

OpenTransact

For interoperability, oscurrency speaks the OpenTransact protocol. For instance, payments can be made from a cell phone.

Lots of web admin tricks

From San Francisco to Athens, Greece, this project has adapted and grown to support moderately sized communities with multiple roles and admin tricks.

Internationalization

Languages supported include Greek, French, Spanish and English.

Authors and Contributors

In 2008, Michael Hartl (@mhartl) created insoshi. The groups feature came from a fork by Ruben Gomez (@yuki). In September 2008, insoshi was forked by oscurrency maintainer Tom Brown (@herestomwiththeweather) and Rich Vazquez (@drapetomaniac). In 2009-2010, heroku support was added by Lee Azzarello (@lazzarello) and Mike Travers (@mtravers). In 2011, internationalization and various admin tools were added by Vasili Revelas (@3point2), Thanasis Peppas (@thanasispeppas) and John Salatas (@jsalatas).
